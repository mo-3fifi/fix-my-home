# Component Structure

* HTML file `*`
    ```sh
    <!DOCTYPE html>
    <html lang="en" dir="ltr">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
      </head>
      <body>
      </body>
    </html>
    ```
* CSS file
    ```sh
    <link rel="stylesheet" href="assets/css/component-name.css">
    ```
* JS file
    ```sh
    <script src="assets/js/component-name.js"></script>
    ```
